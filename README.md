# @cmu-sei/sei-design-system

Vue Components and Tailwind CSS for the SEI Design System.

## Recommended IDE Setup

We recommend the use of [Volar](https://marketplace.visualstudio.com/items?itemName=johnsoncodehk.volar) with VS Code.

## Copilot Instructions File Usage

This project uses Copilot instructions files to guide code generation and modification, especially for unit tests. For example, the file `.github/instructions/unit-testing-guidelines.instructions.md` provides best practices and requirements for writing and updating unit tests in `src/components/**/*.{spec.js,spec.ts}`. Please refer to these instructions when contributing or generating code with Copilot to ensure code consistency and quality.

For more information on configuring custom Copilot instructions for your repository, see the [GitHub documentation](https://docs.github.com/en/enterprise-cloud@latest/copilot/how-tos/configure-custom-instructions/add-repository-instructions?tool=vscode).
